## 1. 프로젝트 주제

내가 만들 프로그램의 주제:

**증상과 나이에 따른 상비약 추천 프로그램**

---

## 2. 문제 상황

이 프로그램이 해결하려는 문제를 설명하세요.

예) 사용자가 현재 증상과 나이를 입력하면, 조건에 맞는 상비약을 추천해 주고 싶다.

내 설명:

사용자가 몸이 아플 때 어떤 상비약을 먹어야 하는지 쉽게 알 수 있도록 도와주고 싶다. 증상과 나이를 입력하면 적절한 약을 추천하고, 어린이가 복용할 수 없는 약은 경고 메시지를 출력하도록 만들고자 한다.

---

## 3. 입력-처리-출력 구조

### 입력

사용자가 입력하거나 프로그램이 사용할 데이터는 무엇인가?

* 현재 증상
* 사용자 나이
* 약 정보가 저장된 2차원 리스트

### 처리

입력된 데이터를 어떤 규칙이나 알고리즘으로 처리할 것인가?

* 2차원 리스트를 반복문으로 탐색한다.
* 입력한 증상과 같은 증상을 치료하는 약을 찾는다.
* 사용자가 12세 미만이고 어린이 복용이 불가능한 약이면 경고 메시지를 출력한다.

### 출력

프로그램이 최종적으로 무엇을 출력하는가?

* 추천 약 이름
* 복용 방법 또는 주의 메시지

---

## 4. 2차원 리스트 설계

내 프로그램에서 사용할 2차원 리스트의 이름:

medicine_data

```python
medicine_data = [
    ["타이레놀", "두통", "하루 3번, 1~2정", "Y"],
    ["베아제", "소화불량", "하루 3번, 1정", "N"],
    ["화이투벤", "감기", "하루 3번, 2캡슐", "Y"],
    ["지르텍", "알레르기", "하루 1번, 1정", "N"]
]
```

### 한 행의 의미

약 1개의 정보를 저장한다.

### 각 열의 의미

| 열 번호 | 의미           | 예시          |
| ---: | ------------ | ----------- |
| 0번 열 | 약 이름         | 타이레놀        |
| 1번 열 | 치료 증상        | 두통          |
| 2번 열 | 복용 방법        | 하루 3번, 1~2정 |
| 3번 열 | 어린이 복용 가능 여부 | Y           |

---

## 5. 함수 설계

| 함수 이름               | 역할          | 입력값           | 반환값 또는 출력         |
| ------------------- | ----------- | ------------- | ----------------- |
| search_medicine     | 증상에 맞는 약 검색 | 약 데이터, 증상, 나이 | 약 정보 또는 "어린이불가"   |
| print_medicine_info | 약 정보 출력     | 약 정보          | 추천 약 이름과 복용 방법 출력 |
| main                | 전체 프로그램 실행  | 없음            | 사용자 입력 및 결과 출력    |

## 6. 사용할 제어구조

### 조건문을 사용할 부분

예) 사용자의 예산보다 가격이 낮은 항목만 추천한다.

내 계획:

* 사용자가 12세 미만인지 확인한다.
* 어린이 복용이 불가능한 약인지 확인한다.
* 검색 결과가 있는지 없는지 확인한다.

### 반복문을 사용할 부분

예) 2차원 리스트의 모든 항목을 하나씩 검사한다.

내 계획:

* 반복문을 사용하여 모든 약 정보를 순서대로 검사한다.
* 입력한 증상과 일치하는 약을 찾는다.
